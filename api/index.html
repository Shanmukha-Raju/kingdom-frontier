About the Game

Kingdom Frontier is a story-driven RPG where each NPC follows a role in the adventure. Returning to the same NPC changes the conversation based on your past visits.

NPC Roles:
Mira = Merchant, Captain Aldric = Gatekeeper, Elder Thorn = Forest Elder, Shade = Shadow Agent.

This interface is built to stay compatible with Unity later: the backend is API-driven, and Unity can call the same endpoints with C#.
